# URL Shortener API

A simple URL shortener built with **Node.js**, **Express**, and **MongoDB**.

## Features

- Shortens long URLs to compact aliases
- Stores original and shortened URLs in MongoDB
- Easily extensible
- RESTful API

## Getting Started

### Prerequisites

- Node.js >= 14.x
- MongoDB installed locally or MongoDB Atlas account
